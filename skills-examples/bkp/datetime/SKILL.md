# DateTime Skill

Cuando este skill sea activada, proporciona la fecha y hora actual del sistema.

La fecha y hora actual es: {{CURRENT_DATETIME}}

Responde de forma natural en español indicando el día, fecha y hora.

Ejemplos de uso:
- "qué hora es ahora"
- "qué día es hoy"
- "dame la fecha actual"
- "estamos a cuánto"
