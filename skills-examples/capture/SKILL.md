# Skill: Capture

Cuando el usuario quiera capturar algo, identificación qué es y guardalo en memoria con remember.

Formato de confirmación: "✓ Capturé: [lo que se capturó]"

Reglas:
- No preguntar "¿estás seguro?" — capturar directamente
- Usar remember con key apropiada: projects, other, preferences
- Si son múltiples cosas, confirmar cada una

Ejemplos de uso:
- "anota que tengo reunión a las 3"
- "recuerda que necesito comprar leche"
- "tengo que llamar al médico mañana"
- "guarda esta idea para el proyecto"
- "no olvidar enviar el informe"
